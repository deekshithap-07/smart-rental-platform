# Rentora - Smart Rental Marketplace

## Project Overview
Rentora is a peer-to-peer rental marketplace designed for fashion, lifestyle products, accessories, and gadgets. The application enables users to rent products for short-term use while allowing owners to list their own items and generate income.
The platform promotes affordability, sustainability, and efficient utilization of products that are otherwise used only occasionally.

## Problem Statement
Many fashion and lifestyle products are expensive and often remain unused after a few uses. Purchasing such items is not always economical for users.

Rentora addresses this challenge by providing a rental platform where users can access products at affordable rates and owners can earn money by renting out underutilized items.

## Application Workflow

1. User creates an account or logs in.
2. User browses products by category.
3. User selects a product and rental duration.
4. Security deposit is applied during booking.
5. Rental request is confirmed.
6. Product owner receives rental request.
7. Product is returned after the rental period.
8. Security deposit is refunded after successful return.
